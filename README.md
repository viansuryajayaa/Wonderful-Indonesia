# Wonderful Indonesia

Wonderful Indonesia is a simple tourism-inspired landing page that introduces the beauty, diversity, and cultural richness of Indonesia through a clean and engaging web interface.

This project was built as a front-end practice project using HTML, CSS, and JavaScript, with a focus on layout structure, visual presentation, and responsive navigation.

## Overview

The website is designed as a single-page experience with multiple sections that highlight Indonesia’s identity, travel appeal, and cultural diversity. It includes a video-based hero section, informational content, image galleries, featured destinations, and a simple team/contact section.

## Features

- Responsive navigation menu
- Fixed navbar with scroll effect
- Full-screen video hero section
- Smooth single-page scrolling
- About section with introductory content
- Highlight section for key Indonesian characteristics
- Image gallery
- Featured destination/blog cards
- Team section
- Footer with contact and sitemap information

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **jQuery**

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/viansuryajayaa/Wonderful-Indonesia.git
   ```

2. Open the project folder:
   ```bash
   cd Wonderful-Indonesia
   ```

3. Run the project by opening `index.html` in your browser.

For a better development experience, you can use **Live Server** in VS Code.
